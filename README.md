# 🎬 Movie Recommender System

This is a **content-based movie recommender system** built using Machine Learning and deployed using **Streamlit**. It takes a movie title as input and recommends similar movies based on textual features.

---

## 🚀 Features

- Recommends top similar movies using vector similarity.
- Clean and interactive **Streamlit** web interface.
- Based on movie metadata from Kaggle dataset.
- Fast and efficient using **TF-IDF / Count Vectorization** and **cosine similarity**.

---

## 🛠️ Tech Stack

| Tool         | Description                           |
|--------------|---------------------------------------|
| `Python`     | Core language                         |
| `pandas`     | Data manipulation                     |
| `scikit-learn` | Machine Learning tools & vectorization |
| `Streamlit`  | Web interface                         |
| `Kaggle`     | Source of dataset                     |

---

## 📁 Dataset

- Used a movie dataset from [Kaggle](https://www.kaggle.com/datasets).
- Contains movie titles, genres, keywords, cast, crew, and more.
- Combined selected features into a single "tags" column for recommendation.
